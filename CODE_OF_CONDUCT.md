# Code of conduct

This project adopts version 2.1 of the Contributor Covenant. The full
text is maintained at its canonical source:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

Concerns about conduct in this project's repository, issues, pull
requests, or any other project communication channel may be reported
to <aah@roarmouse.org>. Reports will be reviewed and responded to
promptly and confidentially.
